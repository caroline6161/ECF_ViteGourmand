# 👨‍🍳 Vite & Gourmand - Application Web de Commande de Menus (Julie & José)

Ce projet est une application web dynamique développée pour l'entreprise de traiteurs **Vite & Gourmand** dans le cadre de l'ECF Développeur Web et Web Mobile. Elle permet aux visiteurs de consulter les menus de saison et de passer commande en ligne de manière sécurisée.

## 🚀 Fonctionnalités Clés
- **Catalogue & Détails des Menus** avec filtres multicritères (MySQL).
- **Espace d'Administration & Employé** complet (Gestion des statuts, blocage de comptes).
- **Statistiques Hybrides (MySQL / NoSQL MongoDB)** : Graphique dynamique de comparaison des ventes via Chart.js et calcul de Chiffre d'Affaires avec filtres de dates.
- **Pages Réglementaires** conformes RGPD & RGAA (Mentions Légales, CGV, À Propos, Contact).

## 🛠️ Stack Technique
- **Front-end :** HTML5, CSS3, Bootstrap 5, JavaScript (Chart.js)
- **Back-end :** PHP (Architecture modulaire, requêtes préparées avec PDO)
- **Bases de données :** MySQL (Relationnel) & MongoDB (NoSQL pour l'historique des ventes)

### 1. Prérequis
- Avoir installé [XAMPP](https://www.apachefriends.org/) (Version PHP 8.x recommandée).
- Git configuré sur votre machine.

### 2. Clonage du projet
Clonez le dépôt dans votre répertoire `htdocs` de XAMPP :
